# ![Home Assistant Logo](https://cdn.jsdelivr.net/gh/pauby/ChocoPackages@caf75f1/icons/homeassistant.png "Home Assistant Logo") [Home Assistant](https://community.chocolatey.org/packages/homeassistant)

The Home Assistant package has been deprecated as it no longer runs under native Windows. You can install it with `pip3 install homeassistant` but once started with `py -m homeassistant --open-ui` you get the message:

```
Home Assistant only supports Linux, OSX and Windows using WSL
```

All package versions have been unlisted.
